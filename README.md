# Project Gallery

Простое веб-приложение для управления проектами.

## Установка и запуск:

### Backend:
```sh
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend:
```sh
cd frontend
npm install
npm start
```

## По умолчанию:

### Backend работает на http://127.0.0.1:5000
### Frontend работает на http://localhost:3000
